What It Does

  • Grid of projects with category filters (ads, reels, longform)
    • A central hub for my cinematic edits with category filters, lite YouTube embeds, and clean case-study notes (tools, credits, before/after).

Why I Built It

I needed a fast way to showcase edits without heavy players, and to add context (tools, roles, before/after) that helps clients choose the right service—plus a clear “Book a Free Edit” CTA.

Tech Stack

  • HTML (or Next.js)
  • HTML + Tailwind (dark/glass UI)
  • Lite YouTube embeds with vanilla JS
  • JSON content for projects (title, category, ytid, caption)
  • Modal/lightbox for case-study notes
  • Formspree (or Node/Express) contact
  • Netlify/Vercel deploy; FFmpeg for thumbnails

How to Run

  1. Add thumbnails in assets/ and collect YouTube IDs
  2. Update projects.json ({ title, category, ytid, caption, thumb })
  3. Open the page locally (static) or run a tiny server
  4. Deploy to Netlify/Vercel and set meta/OG image
