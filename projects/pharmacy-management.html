Pharmacy Management System

An inventory and billing system for pharmacies — manage medicines, sales, and suppliers with automated stock and expiry alerts.

React Node.js MongoDB Express
View Code Live Demo
Dashboard / POS Screenshot

What It Does

  • Medicine inventory with batch, expiry, and supplier info
  • Barcode-based search and quick-billing POS interface
  • Sales reporting and daily summaries
  • Purchase orders and stock refill tracking
  • Role-based access for admin and cashier

Why I Built It

I wanted to apply the MERN stack to a real-world retail problem — keeping pharmacy operations organized and efficient.

Tech Stack

  • React.js (frontend)
  • Node.js + Express.js (backend)
  • MongoDB (database)
  • Tailwind (UI)
  • JWT authentication

How to Run

  1. Clone repo and install dependencies
  2. Setup MongoDB URI in .env
  3. Run backend and frontend concurrently
  4. Login with demo admin credentials
