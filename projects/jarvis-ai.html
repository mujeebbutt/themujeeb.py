JARVIS AI Assistant

A voice-controlled desktop assistant that launches apps, fetches info, and automates workflows — talks to you like a real helper.

Python Speech Recognition Automation OS Control
View Code Demo Video

What It Does

  • Wake-word & push-to-talk activation.
  • Natural speech commands with intent parsing.
  • Open apps, search web, tell time, play media, control volume/brightness.
  • Speaks responses with pyttsx3 TTS.
  • Logs queries/responses to SQLite for history & analytics.
  • Pluggable “skills” router to add custom commands in minutes.

Why I Built It

I wanted a practical daily driver for my machine — faster than clicking, flexible like a CLI, and friendly with voice feedback.

Try: "open vs code" · "what's the time" · "play study playlist"

Tech Stack

  • Python 3.10
  • speech_recognition + PyAudio
  • pyttsx3 (TTS)
  • Optional: OpenAI/LLM for NLP
  • OS-specific automation libs

How to Run

  1. Create virtualenv
  2. Install requirements.txt
  3. Add API keys (if any)
  4. Run main.py and say the wake word

Tip: map a global hotkey to push-to-talk when mic access is noisy.

Highlights & Learnings
