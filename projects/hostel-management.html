Hostel Management System (WIP)

Resident & room management with fee tracking, attendance, and issue tickets — currently in progress.

Python Django PostgreSQL
View Code Live Demo
Project hero image / GIF

What It Does

  • Rooms, beds, allocation & availability
  • Resident profiles, guardian contacts
  • Monthly fee invoices & payment status
  • Attendance & incident logging

Why I Built It

To generalize the LMS/Pharmacy patterns into a structured multi-entity system with robust permissions and reports.

Tech Stack

  • Django
  • DRF (optional for API)
  • PostgreSQL
  • Tailwind UI

How to Run

  1. Set up virtualenv
  2. Install requirements
  3. Configure DB & migrate
  4. Run dev server
