Mehwish Butt — Online Tutor

A modern teaching platform: Google Sign-In for parents/students, clean booking flow, and an LMS for grades, attendance, and fee tracking — built with HTML/Tailwind, Python, and Node.js.

What It Does

  • Service catalog with grade/subject filters
  • Google Sign-In (parents & students) for instant, secure access.
  • LMS dashboard: classes, students, attendance, assignments, and gradebook.
  • Fees module: invoices, status (paid/unpaid), due reminders, and receipts.
  • Booking flow for new lessons with calendar time slots.
  • Uploads: share notes/homework; view history per student.
  • Responsive UI (Tailwind) with accessible forms and clear CTAs.

Why I Built It

I wanted a simple, secure tutoring site that goes beyond booking — with Google login, a lightweight LMS for grades, and a fees module parents actually understand. The stack lets me iterate quickly while keeping the UX clean.

Tech Stack

  • Vanilla HTML/CSS/JS
  • Stripe Checkout (optional)
  • Zoom API (optional)
  • Formspree/EmailJS for emails

How to Run

  1. Set env vars in .env: GOOGLE_CLIENT_ID, GOOGLE_CLIENT_SECRET, OAUTH_CALLBACK_URL, DB_URI, SESSION_SECRET.
  2. Install and start API (Node): npm install && npm run dev (or npm start)
  3. (Optional) Enable Python service (reports): pip install -r requirements.txt && python reports.py
  4. Open http://localhost:3000 (or your configured port) and sign in with Google.

Highlights & Learnings

  • Frontend: HTML + Tailwind (dark/glass UI)
  • Auth: Google OAuth 2.0 (Passport.js or Google Identity)
