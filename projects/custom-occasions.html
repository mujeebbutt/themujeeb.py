Custom Occasion Sites (Birthdays & Proposals)

Personalized microsites with animations, countdowns, galleries, and RSVP — made to create moments.

HTML Tailwind JavaScript Airtable
View Code Live Demo
Project hero image / GIF

What It Does

  • Hero animation + countdown timer
  • Photo gallery & memory lane section
  • Interactive RSVP with guest messages
  • Optional: private passcode access

Why I Built It

A fun way to blend design and emotion — and a quick productized service I can ship in a day.

Tech Stack

  • Tailwind CSS
  • Vanilla JS
  • Airtable/Google Sheets backend (no-code)
  • Netlify Forms (optional)

How to Run

  1. Duplicate template
  2. Swap images & text
  3. Connect RSVP storage
  4. Deploy to Netlify/Vercel
